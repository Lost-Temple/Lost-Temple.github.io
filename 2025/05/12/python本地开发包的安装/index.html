python本地开发包的安装

发表于 2025-05-12 分类于 Python ， 基础

pip install -e .

pip install -e .的作用是以“可编辑模式”安装当前目录下的Python包。这是Python项目开发过程中常用的操作。

📌 具体解释如下：

  • pip：Python 的包管理工具。
  • install：表示你要安装某个包。
  • -e 或 --editable：表示“可编辑”模式（editable mode）。
  • .：指当前目录，意思是安装当前目录下的包。

🔧 典型应用场景：

你在 my_project目录下执行：

这时，Python 会：

  1. 读取 setup.py 来安装这个包。
  2. 创建一个指向源代码目录的符号链接（而不是复制代码）。
  3. 你修改 my_package/ 中的代码后，不需要重新安装包，修改立刻生效。

✅ 适合：

  • 正在开发或调试一个 Python 包。
  • 希望修改代码后立即看到效果。

🆚 普通安装 (pip install .) 区别：

命令 是否复制代码 修改源码是否自动生效 场景
pip install . ✅ 是 ❌ 否 生产环境
pip install -e . ❌ 否 ✅ 是 开发调试阶段
